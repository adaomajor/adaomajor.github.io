What is MPF?

MPF (Minimalist PHP Framework) is a lightweight, flexible, open-source PHP framework for building modern web applications. It embraces a clean MVC approach, minimal dependencies, and fast developer experience.

  • Minimal setup, fast routing, and simple views
  • MVC structure that stays out of your way
  • Open source (MIT) — easy to extend and maintain
